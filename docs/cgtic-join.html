发酵工程师

@ 研发部

岗位职责
  1. 负责细胞与基因治疗载体（AAV、AD、LV、plasmid、mRNA）发酵工艺小试、中试开发及优化；
  2. 辅助新型发酵方法、病毒制备方法的工艺开发及放大研究；
  3. 负责发酵专员的培训、考核及日程安排；
  4. 配合发酵工艺从研发到GMP的工艺转移；
  5. 负责病毒制备研发车间的管理及规则制定。
任职要求
  1. 大专及以上学历， 医学，药学，生命学相关专业
  2. 5-10年哺乳动物细胞从小试到中试发酵工艺开发经验；
  3. 熟练使用不同规格生物反应器；
  4. 有病毒载体或疫苗生产经验者优先；
  5. 工作认真踏实，具有团队协作精神，可接受法定假日的调休；
  6. 实验操作能力强，实验习惯良好。

QC经理

@ 质量部

岗位职责
  1. 负责产品安全性指标如微生物限度、无菌、内毒素、支原体的方法开发与验证。
  2. 负责并组织人员对标准物质、试剂、试液、培养基、检定菌进行管理。
  3. 负责并组织人员对试验室设备仪器、器具的管理和维护。
  4. 负责并组织人员对分析设备进行必要的验证、确认。
  5. 负责并组织人员对 OOS、OOT 进行原因彻底调查、纠正预防。
  6. 负责并组织人员对 QC 相关文件进行维护、修订、起草。
  7. 负责并组织和指导人员进行检验方法确认、验证。
  8. 负责并组织人员接受新产品分析方法的转移。
  9. 负责根据试验室新增检验项目、设备运行情况，配备必要的试验设施、工具书、标准物质。
  10. 支持公用系统检验、原辅料检验及常规产品放行检验；
任职要求
  1. 药学、细胞生物学等相关专业，本科及以上学历，熟练掌握分子生物学实验技能。
  2. 工作年限5年以上，其中具备 QC 管理岗位 2 年以上。具有生物制品公司工作经验。
  3. 熟悉 GMP、GMP 实施指南、中国药典和 USP 等相关法律法规和指南。
  4. 具有制定和维护 QC 相关 GMP 文件的经验。
  5. 熟悉 PCR、ELISA、HPLC、FACS 以及常规理化和微生物等相关实验操作经验或管理经验。
  6. 据有检验方法学验证经验，担任过方法验证方案起草，实施、报告工作。
  7. 具有检验设备确认验证经验，担任过起草方案，实施、报告工作。
  8. 具有较强的 OOS、OOT 调查和分析解决问题的经验和能力。

QA经理

@ 质量部

岗位职责
  1. GMP质量体系建立、维护、审核，偏差调查和处理，OOS/OOT/CAPA关闭，风险评估报告等；
  2. 药协调公司内外的审计、检查，公司项目日常运营中的合规性支持；；
  3. 文件体系管理；
  4. 人员培训大纲及计划的制定、批准、变更，培训报告的批准，监督生物药中心质量体系知识的培训，监督及督促实施药品生产管理规范、法规的培训；监督公司各类人员的GMP知识和药品质量意识的培训工作完成情况并完成考核。
  5. 物料和供应商管理；
  6. 生产管理，审核、批准研发与生产的工艺技术转移报告、生产指令、对中间体、半成品及成品进行放行、不合格品的处理程序等；
  7. 检验管理，审核检验方法转移报告，中间产品、成品的检验符合要求和质量标准，并对产品放行，审核稳定性研究数据，批准稳定性研究报告；
  8. 验证管理，年度验证主计划以及产品生产、清洁、检验相关的验证与确认的审批；
  9. 生物药原液及制剂生产现场监控工作。
任职要求
  1. 药学及相关专业本科及以上学历；
  2. 熟悉GMP规范，了解中美欧法规；
  3. 具有5年以上制药企业QA管理工作经验，其中2年以上的生物药QA管理工作经验；
  4. 具有基因、疫苗生产相关工作经验者优先；
  5. 具有较强的沟通、协调、管理、计划与执行能力。

生产上游工程师

@ 生产部

岗位职责
  1. 参与上游生产设备的验收验证工作，生产相关的文件起草。
  2. 按照设备标准操作规程进行设备操作，并及时记录设备运行日志及设备状态信息
  3. 依据生产计划和生产指令进行细胞培养复苏、扩增、生产、转染，收获等，也包括培养工艺优化、中试放大。
  4. 依据生产计划和生产指令进行大肠杆菌发酵，按照种子制备、菌种发酵的工序标准操作规程，接收菌种和培养基， 进行菌种活化、传代、发酵、菌体收集、菌体洗涤等工作。
  5. 负责相关设备日常维护与管理；
  6. 其他临时性工作。
任职要求
  1. 大专或以上学历，生物工程、生物制药、制药工程或相关专业；
  2. 有2-3年以上HEK293或CHO细胞工艺开发及GMP中试放大相关工作经验优先
  3. 对细胞培养技术较为了解，能够发现细胞培养项目开发中遇到的各类问题；
  4. 拥有200L规模以上细胞培养经验者优先；
  5. 有一定的中文文献检索、阅读以及实验设计能力；
  6. 工作认真、踏实、责任心强、有较强团队合作意识。
